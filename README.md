### 🎯 Problem Statement
Why were high-performing employees with strong performance leaving the company quickly? 

### 🛠 Tools & Technologies
* **Python:** Used for advanced data cleaning and preprocessing. 
* **Power BI:** Built the interactive visualization layer. 
* **Advanced DAX:** Implemented **Time Intelligence functions** for trend analysis. 
* **HR Metrics:** Calculated key performance indicators to measure workforce health. 

### 💡 Business Impact
***Identified Attrition "Hotspots":** Pinpointed exactly which departments and salary ranges had the highest turnover. 
* **Retention Strategies:** Provided data-driven insights that allowed HR to implement targeted strategies to minimize turnover costs.
